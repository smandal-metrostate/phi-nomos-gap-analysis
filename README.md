# PHI-Nomos: Retroactive BAA Termination

An interactive Nomos modular norm model for a HIPAA PHI gap-analysis case study,
built in the visual and reasoning style of the original Modular Norms toolchain
([robinagandhi/modularnorms](https://github.com/robinagandhi/modularnorms)).

**Live demo:** https://<your-username>.github.io/<your-repo>/

Applied to 45 CFR §164.308(b)(1) (Business Associate Contracts), §164.410/§164.404
(Breach Notification), and §164.504(e)(2)(ii)(J) (Disposition of PHI at Termination).

- Hover any node for the underlying legal text
- Click a situation box to toggle it satisfied / not satisfied
- Press **Solve** to propagate compliance through the model and generate a gap register

Reasoning runs entirely client-side — no external norm server required.
